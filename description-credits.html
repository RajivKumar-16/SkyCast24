About SkyCast24

SkyCast24 is a modern, fast, and responsive weather forecast web app that provides accurate weather information using real-time data. It is designed to be lightweight and visually intuitive, making weather checking a seamless experience.

Key Features

  • Location-Based Weather: Detects your current location for instant weather data.
  • City Name Search: Search any city to get precise forecasts.
  • Live Clock: Auto-refreshing current time display.
  • Hourly & 7-Day Forecast: Clean, card-style data display.
  • Day/Night Icons: Weather icons adjust based on time of day.

Technologies Used

  • HTML5 + CSS3 – Layout and styling
  • JavaScript (Vanilla) – App logic and interactivity
  • Open-Meteo API – Live weather data source
  • OpenStreetMap Nominatim API – Geocoding for city names

Credits

  • Weather Data: Open-Meteo
  • Geocoding: OpenStreetMap Nominatim
  • Icons: Adapted from weather-icons
